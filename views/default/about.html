Waterdrops: Long story sort...

I created Waterdrops for all hydrologist who seek a simple and effective way to store / display their hydrological results.

The main idea is Waterdrop to create a social network of Watersheds/Catchments.

I am not a computer scientist or web developer geek ... All Waterdrop source code created with hard work in after hours by realizing "request" and "response" meaning ...

All code is built using Python 2.7 (web2py framework , folium library for map display, pandas for time series analysis, mpld3 library for plots).
